# CBL-OpenSim
## Infant head-neck musculoskeletal modeling pipeline 
  _Boise State University, 2026_

This repository contains MATLAB, OpenSim, and supporting workflow tools for processing infant head-neck modeling data, including motion capture, ground reaction force processing and/or simulation, inverse dynamics, static optimization, and model analysis.

### Data Privacy
*IMPORTANT*

Certain functions/scripts in this repository are designed to operate on subject-level CT data. Thus, they may require the input of CT data in order to function properly. Do ***NOT*** store any deidentified, subject-level data globally. The gitignore will explicitly block commits for certain file types but is not a fool-proof catch all. Never publicly share any data related to human subjects without express permission via an authorized institutional review board and subject or next-of-kin.
